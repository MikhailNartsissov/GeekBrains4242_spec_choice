# Итоговая контрольная работа по основному блоку
## Описание задачи

Написать программу, которая из имеющегося массива строк 
формирует новый массив из строк, длина которых меньше, либо равна 3 символам. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения 
алгоритма. При решении не рекомендуется пользоваться коллекциями, лучше обойтись 
исключительно массивами.

## Решение

Для получения исходных данных используется заранее созданный пустой массив строк [str] initialArray.
С помощью метода Console.ReadLine() у пользователя запрашиваются данные в виде строки, состоящей из 
элементов массива, разделенных запятыми, после чего, с помощью метода Split(',') введенная строка 
делится на элементы массива и с помощью методов Select(x => x.ToString().Trim()).ToArray() 
наш исходный массив заполняется введенными элементами, очищенными от ведущих и хвостовых пробелов.

### Метод LessThanFour

В этом методе создается результирующий массив строк. Далее циклом

возвращаемый методом.

### Метод PrintArray

В этом методе циклом for (int findex = 0; findex < farr.Length - 1; findex++) выполняется перебор элементов 
принятого массива и сравнение слещующего за текущим элемента с null. Если проверка выполняется, то элемент 
принятого массива печатается с добавлением запятой и пробела в конце. Если нет, то элемент принятого массива 
печатается с добавлением квадратной скобки и цикл прерывается. Таким образом подавляется вывод пустых элементов 
результирующего массива.
